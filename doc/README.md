## 学习路径建议

### 初学者路径

1. 阅读 **项目结构分析** 了解整体框架
2. 阅读 **使用说明** 的安装和快速开始部分
3. 运行示例代码（samples目录）
4. 阅读 **架构图解** 理解核心流程
5. 逐步学习 **使用说明** 的高级功能

### 中级用户路径

1. 深入学习 **使用说明** 的所有章节
2. 阅读 **开发指导** 的核心组件开发部分
3. 参考 **架构图解** 理解内部机制

## 示例代码索引

项目 `samples/` 目录包含70个示例，以下是重点示例：

### 基础示例
- `sigsmacross/` - SMA交叉策略（最简单的入门示例）
- `data-pandas/` - 使用Pandas数据
- `data-multitimeframe/` - 多时间框架

### 策略示例
- `multidata-strategy/` - 多数据源策略
- `bracket/` - Bracket订单
- `oco/` - OCO订单
- `stoptrail/` - 追踪止损

### 优化示例
- `optimization/` - 参数优化

### 实盘示例
- `ibtest/` - Interactive Brokers实盘
- `oandatest/` - Oanda实盘

### 高级功能
- `commission-schemes/` - 佣金方案
- `slippage/` - 滑点
- `timers/` - 定时器
- `tradingcalendar/` - 交易日历
