## 5. 订单执行流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              订单执行流程                                    │
└─────────────────────────────────────────────────────────────────────────────┘

策略层
┌──────────────────────────────────────────────────────────────────────────┐
│  Strategy.next()                                                         │
│  ┌────────────────────────────────────────────────────────────────────┐  │
│  │  if signal:                                                         │  │
│  │      order = self.buy()                                             │  │
│  └────────────────────────────────────────────────────────────────────┘  │
└───────────────────────────────────┬──────────────────────────────────────┘
                                    │
                                    ▼
订单创建
┌──────────────────────────────────────────────────────────────────────────┐
│  Order Creation                                                          │
│  ┌────────────────────────────────────────────────────────────────────┐  │
│  │  1. 创建 Order 对象                                                 │  │
│  │  2. 设置订单类型 (Market/Limit/Stop/etc)                           │  │
│  │  3. 设置价格、数量、有效期等参数                                    │  │
│  │  4. 状态: Created                                                   │  │
│  └────────────────────────────────────────────────────────────────────┘  │
└───────────────────────────────────┬──────────────────────────────────────┘
                                    │
                                    ▼
订单提交
┌──────────────────────────────────────────────────────────────────────────┐
│  Broker.submit(order)                                                    │
│  ┌────────────────────────────────────────────────────────────────────┐  │
│  │  1. 验证订单有效性                                                  │  │
│  │  2. 检查保证金要求                                                  │  │
│  │  3. 状态: Submitted                                                 │  │
│  │  4. 加入订单队列                                                    │  │
│  └────────────────────────────────────────────────────────────────────┘  │
└───────────────────────────────────┬──────────────────────────────────────┘
                                    │
                                    ▼
订单接受
┌──────────────────────────────────────────────────────────────────────────┐
│  Broker.accept(order)                                                    │
│  ┌────────────────────────────────────────────────────────────────────┐  │
│  │  1. 最终验证                                                        │  │
│  │  2. 状态: Accepted                                                  │  │
│  │  3. 准备执行                                                        │  │
│  └────────────────────────────────────────────────────────────────────┘  │
└───────────────────────────────────┬──────────────────────────────────────┘
                                    │
                                    ▼
订单撮合
┌──────────────────────────────────────────────────────────────────────────┐
│  Broker._match_order(order)                                              │
│  ┌────────────────────────────────────────────────────────────────────┐  │
│  │  Market Order:                                                      │  │
│  │    - 使用当前bar的收盘价（或开盘价，取决于模式）                    │  │
│  │    - 立即成交                                                       │  │
│  │                                                                      │  │
│  │  Limit Order:                                                       │  │
│  │    - 检查价格是否触及限价                                           │  │
```
