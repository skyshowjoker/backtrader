# Tushare 数据对接指南

## 数据概况

你的数据文件夹包含:

### 1. 日线数据
- **文件**: `stock_daily.parquet`
- **大小**: 1.29 GB
- **记录数**: 14,099,409 条
- **时间范围**: 2009-01-05 至今
- **数据字段**: 31 个字段
  - 基础行情: open, high, low, close, volume, amount
  - 复权因子: adj_factor
  - 涨跌幅: pct_chg
  - 估值指标: pe, pb, ps, total_mv
  - 其他: turnover_rate, suspend_timing, is_st, listed_days 等

### 2. 分钟线数据
- **文件夹**: stock_1min, stock_15min, stock_30min, stock_60min
- **文件数量**: 每个文件夹约 5769 个文件(每只股票一个文件)
- **数据字段**: 8 个字段
  - ts_code, open, high, low, close, vol, amount, adj_factor

## 快速开始

### 1. 基本使用

```python
from tushare_data_loader import TushareDataLoader
import backtrader as bt

# 创建数据加载器
loader = TushareDataLoader('/Users/mac/Downloads/行情数据')

# 加载日线数据
data = loader.load_daily_data(
    ts_code='000001.SZ',        # 股票代码
    start_date='2020-01-01',    # 开始日期
    end_date='2023-12-31',      # 结束日期
    adj_type='hfq'              # 后复权
)

# 添加到回测引擎
cerebro = bt.Cerebro()
cerebro.adddata(data)
```

### 2. 复权类型说明

- **hfq (后复权)**: 价格 × 复权因子,保持价格连续性
- **qfq (前复权)**: 价格 × 复权因子 / 最新复权因子,当前价格不变
- **none (不复权)**: 使用原始价格

### 3. 加载分钟线数据

```python
# 加载15分钟线
data_15min = loader.load_minute_data(
    ts_code='000001.SZ',
    freq='15min',              # '1min', '15min', '30min', '60min'
    start_date='2023-01-01',
    adj_type='qfq'
)
```

### 4. 加载多只股票

```python
# 多股票策略
stocks = ['000001.SZ', '000002.SZ', '600000.SH']
for stock in stocks:
    data = loader.load_daily_data(stock, '2022-01-01', '2023-12-31')
    cerebro.adddata(data)
```

## 完整示例

### 日线策略回测

```python
from datetime import datetime
import backtrader as bt
from tushare_data_loader import TushareDataLoader

class MyStrategy(bt.Strategy):
    def __init__(self):
        self.sma = bt.ind.SMA(period=20)

    def next(self):
        if not self.position:
            if self.data.close[0] > self.sma[0]:
                self.buy()
        else:
            if self.data.close[0] < self.sma[0]:
                self.sell()

# 初始化
cerebro = bt.Cerebro()
loader = TushareDataLoader()

# 加载数据
data = loader.load_daily_data('000001.SZ', '2020-01-01', '2023-12-31')
cerebro.adddata(data)

# 添加策略和分析器
cerebro.addstrategy(MyStrategy)
cerebro.addanalyzer(bt.analyzers.SharpeRatio, _name='sharpe')
cerebro.addanalyzer(bt.analyzers.DrawDown, _name='drawdown')

# 运行
cerebro.broker.setcash(100000)
results = cerebro.run()

# 查看结果
print(f"收益率: {(cerebro.broker.getvalue() / 100000 - 1) * 100:.2f}%")
```

### 参数优化示例

```python
# 参数优化
cerebro = bt.Cerebro()
loader = TushareDataLoader()
data = loader.load_daily_data('000001.SZ', '2020-01-01', '2023-12-31')
cerebro.adddata(data)

# 添加优化策略
cerebro.optstrategy(
    MyStrategy,
    period=range(10, 31, 5),
    threshold=[0.3, 0.5, 0.7]
)

# 运行优化
results = cerebro.run(maxcpus=4)
```

## 高级功能

### 1. 自定义数据字段

如果需要使用数据中的其他字段(如 pe, pb 等):

```python
class ExtendedData(bt.feeds.PandasData):
    lines = ('pe', 'pb', 'adj_factor')
    params = (
        ('pe', 'pe'),
        ('pb', 'pb'),
        ('adj_factor', 'adj_factor'),
    )

# 在加载器中修改 PandasData 类
data = ExtendedData(dataname=df)
```

### 2. 数据重采样

将分钟线转换为其他周期:

```python
# 加载分钟数据
data_15min = loader.load_minute_data('000001.SZ', '15min', '2023-01-01')

# 重采样为日线
cerebro.resampledata(data_15min, timeframe=bt.TimeFrame.Days)
```

### 3. 多时间周期策略

```python
# 同时加载日线和周线
data_daily = loader.load_daily_data('000001.SZ', '2020-01-01')
data_weekly = cerebro.resampledata(data_daily, timeframe=bt.TimeFrame.Weeks)

# 在策略中使用
class MultiTimeframeStrategy(bt.Strategy):
    def __init__(self):
        # 日线数据
        self.daily_sma = bt.ind.SMA(self.data0, period=20)
        # 周线数据
        self.weekly_sma = bt.ind.SMA(self.data1, period=10)
```

## 性能优化建议

### 1. 数据加载优化
- 只加载需要的日期范围
- 避免一次性加载过多股票
- 使用 `preload=False` 按需加载

### 2. 内存管理
```python
# 大数据集使用
cerebro = bt.Cerebro(exactbars=True)  # 精确内存管理

# 或
cerebro.run(preload=False)  # 不预加载数据
```

### 3. 并行优化
```python
# 参数优化时使用多核
results = cerebro.run(maxcpus=4, optreturn=True)
```

## 常见问题

### Q1: 如何查看所有可用股票?
```python
import pandas as pd
df = pd.read_parquet('/Users/mac/Downloads/行情数据/stock_daily.parquet')
stocks = df.index.get_level_values('ts_code').unique()

```

### Q2: 如何处理停牌数据?
```python
# Backtrader 会自动处理,但在策略中可以检查
class MyStrategy(bt.Strategy):
    def next(self):
        # 检查是否有成交
        if self.data.volume[0] > 0:
            # 正常交易
            pass
        else:
            # 可能停牌
            pass
```

### Q3: 如何使用涨跌停价格?
```python
# 在自定义数据源中添加
class ExtendedData(bt.feeds.PandasData):
    lines = ('up_limit', 'down_limit')
    params = (
        ('up_limit', 'up_limit'),
        ('down_limit', 'down_limit'),
    )

# 在策略中使用
class MyStrategy(bt.Strategy):
    def next(self):
        # 检查是否涨停
        if self.data.close[0] >= self.data.up_limit[0]:
            # 涨停,不买入
            pass
```
